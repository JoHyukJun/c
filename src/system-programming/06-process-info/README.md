## q-05

    2. tmpnam("/tmp", "jo")

## a-05

### 1. tmpnam("jo")

    - char *tmpnam(char *s)
        - 임시 파일명을 시스템이 알아서 지정
        - 인자가 있을 경우 해당 인자가 가리키는 곳에 임시 파일명을 저장
        - 인자가 NULL 이면 임시 파일명 리턴
